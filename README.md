XImóveis

Summary
XImóveis é uma plataforma acadêmica voltada para o mercado imobiliário, desenvolvida com o objetivo de integrar tecnologia, organização e acessibilidade.
O sistema permite o cadastro e gerenciamento de imóveis, filtros de busca detalhados e um mapa interativo, oferecendo uma experiência moderna e eficiente tanto para corretores quanto para compradores.

Description
O projeto foi criado como parte de um trabalho acadêmico para aplicar os fundamentos de desenvolvimento front-end utilizando HTML, CSS e JavaScript.
Seu propósito é demonstrar a construção de uma aplicação web funcional e responsiva, capaz de simular um ambiente real de negócios do setor imobiliário.

A proposta do XImóveis é unir boas práticas de design de interface, interatividade e manipulação de dados no navegador, criando uma base sólida para estudos e futuras expansões com back-end e banco de dados.

Dependencies
O XImóveis é construído com tecnologias nativas de front-end, sem frameworks obrigatórios.
Entretanto, recomenda-se o uso das ferramentas abaixo para ambiente de desenvolvimento:

Node.js (para execução local com servidor)

npm ou yarn (para dependências adicionais, se necessário)

Visual Studio Code (ambiente de desenvolvimento sugerido)

Extensão Live Server (para testes locais em tempo real)

How to Run

Clonar o repositório:

git clone https://github.com/usuario/ximoveis.git
cd ximoveis


Executar localmente com servidor estático:

npm install -g serve
serve .


Ou, se estiver utilizando o Visual Studio Code, abrir o projeto e iniciar o Live Server.

Após isso, acesse no navegador:

http://localhost:3000
