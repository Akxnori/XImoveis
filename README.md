# 🏠 XImóveis

XImóveis é uma plataforma web para imobiliárias e corretores anunciarem imóveis de forma simples e transparente.  
O sistema permite visualizar imóveis em um **mapa interativo**, aplicar **filtros de busca** e consultar o **histórico de preços e status** de cada imóvel.  

---

## 🚀 Funcionalidades

- Cadastro e login de corretores/imobiliárias  
- Cadastro, edição e exclusão de imóveis  
- Exibição de imóveis em **mapa interativo**  
- Filtros de busca (preço, localização, tipo)  
- Histórico de imóveis (preços anteriores e status)  
- Sugestão de imóveis semelhantes  
- Área administrativa para gerenciar anúncios  

---

## 🛠️ Tecnologias Utilizadas

- **Front-end:** HTML, CSS, JavaScript  
- **Back-end:** PHP / Node.js (dependendo da escolha da equipe)  
- **Banco de Dados:** MySQL  
- **Mapas:** Google Maps API ou Leaflet.js
